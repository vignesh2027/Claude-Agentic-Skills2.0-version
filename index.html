⚡ How It Works

Right agent.
Every task.

AgentOS uses a 6-step orchestration loop — each step runs in Claude's context the moment you paste a skill.

1

Identify

Classifies the domain, complexity, and which specialist agent is needed based on your task keywords.

2

Decompose

Breaks complex tasks into focused sub-tasks, each matched to a specialist's core strength.

3

Route

Dispatches each sub-task to the right agent with the right skill set — parallel or sequential.

4

Execute

Each agent applies its frameworks, formulas, and sub-agents. Full depth — no hand-waving.

5

Synthesize

All outputs merged into one coherent, structured response with consistent format.

6

Deliver

Executive Summary · Full Output · Confidence Level · Next Steps · Caveats. Always.

🚀 Quick Start

Up in 60 seconds.

Four ways to activate any AgentOS skill — pick the one that fits your workflow.

1

Claude.ai Projects

Create a Project, open Project Instructions, paste any SKILL.md. Every chat in that project activates the agent.

# 1. Open claude.ai → Projects # 2. New Project → Instructions # 3. Paste SKILL.md contents # 4. Start chatting ✓
2

Claude Code CLI

Append any skill to your project's CLAUDE.md. Combine multiple skills to create a custom AI team.

# Single skill cat quant-trader/SKILL.md >> .claude/CLAUDE.md # Full AgentOS (all 100+) cat agentOS-orchestrator/SKILL.md \ >> .claude/CLAUDE.md
3

Claude API (Python)

Pass the SKILL.md contents as the system parameter. Works with any Claude model.

with open("quant-trader/SKILL.md") as f: skill = f.read() client.messages.create( model="claude-sonnet-4-6", system=skill, messages=[{ "role": "user", "content": "Signal on NVDA" }] )
4

Cursor / Windsurf

Paste SKILL.md into your IDE's AI rules file. Works with any Claude-backed IDE extension.

# Cursor .cursor/rules # Windsurf .windsurfrules # Any IDE: paste into # AI custom instructions
